# 울집 사용 가이드 (자녀용)

## 빠른 시작
- 설치: 브라우저에서 울집을 열고 “홈 화면에 추가/설치”를 누르면 앱처럼 열립니다.
- 알림: “내 정보 > 푸시 알림”에서 권한을 허용하면 가족 소식을 바로 받을 수 있습니다.
- 가족 합류: 초대 코드를 받았다면 “내 정보”에 입력해 가족에 참여하세요.

## 내가 할 일 (10초 버전)
- 원터치 안심 신호: 가운데 `+` 버튼 → 출발/귀가/식사/취침/기상 중 선택 → 초록/노랑/빨강으로 상태 공유. 잘못 눌렀다면 5분 안에 되돌리기 버튼을 누를 수 있습니다.
- 약 복용 공유: “내 정보 > 오늘 복용 체크”에서 아침/점심/저녁 버튼을 눌러 기록하세요. 타임라인과 요약에서 가족이 함께 확인합니다.
- 감정 한 줄: 하루 한 번 이모지 + 한 줄 메시지를 남길 수 있습니다. 짧게 적어도 괜찮아요.
- 오늘 질문 답변: 알림이 오면 하루 질문에 답하고 제출하세요. 답변은 모두 제출/마감 전까지 비공개입니다.
- SOS: 위급할 때 SOS를 누르면 2단계 확인 후 가족에게 경고 배너로 전달됩니다.

## 알림·오프라인 팁
- 푸시가 안 오면: 알림 권한 확인 → PWA 설치 여부 확인 → 한 번 재로그인.
- 오프라인이어도 입력은 보관됐다가 연결되면 전송됩니다.

## 안전/프라이버시
- 자동 위치·과도한 수집은 없고, 내가 남긴 신호/약/감정/답변만 가족과 공유됩니다.
