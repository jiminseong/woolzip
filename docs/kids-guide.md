# 울집 사용 가이드 (자녀용)

## 빠른 시작
- 설치: 브라우저에서 울집을 열고 “홈 화면에 추가/설치”를 눌러 앱처럼 사용하세요.
- 알림: “내 정보 > 푸시 알림”에서 권한을 허용하면 가족 소식을 바로 받습니다.

## 안심 신호 보내기
1. 가운데 ‘+’ 버튼을 눌러 출발/귀가/식사/취침/기상 등 상황을 선택하세요.
2. 초록/노랑/빨강 신호로 상태를 알려주세요. 잘못 눌렀다면 5분 안에 취소할 수 있습니다.

## 약 복용 공유
- 부모님 약이 등록되어 있다면 “내 정보 > 오늘 복용 체크”에서 해당 시간대 버튼을 눌러 드세요.
- 타임라인과 요약에서 복용 여부를 확인할 수 있으니, 하루 한 번씩 챙겨 주세요.

## 감정 한 줄
- 하루 한 번, 이모지와 한 줄 메시지로 가볍게 안부를 남길 수 있습니다. 짧게 써도 괜찮아요.

## 가족 초대/합류
- 초대 코드를 받았다면 “내 정보”에서 입력해 가족에 합류하세요.
- 새 가족이 합류하면 타임라인에 참여 이벤트가 표시됩니다.

## 알림 꿀팁
- 푸시가 안 오면: 알림 권한 확인 → PWA 설치 여부 확인 → 한 번 재로그인.
- 늦은 시간 푸시는 최소화되어 있으니 중요한 소식만 알림으로 옵니다.
